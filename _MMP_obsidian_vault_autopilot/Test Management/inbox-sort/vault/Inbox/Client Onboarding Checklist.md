# Client Onboarding Checklist

Standard process for bringing a new client onto the platform. Target: complete within 5 business days.

## Day 1: Kickoff

1. Send welcome email with login credentials
2. Schedule 30-minute onboarding call
3. Create client workspace in project management tool
4. Assign account manager

## Day 2-3: Configuration

1. Import client's existing data (CSV or API sync)
2. Configure custom fields and workflows
3. Set up user accounts for their team (up to 10 seats)
4. Enable SSO if enterprise tier

## Day 4: Training

1. Conduct live training session (60 minutes, recorded)
2. Share documentation portal link
3. Create support channel in Slack or Teams

## Day 5: Go-Live

1. Verify all integrations are working
2. Send go-live confirmation email
3. Schedule 2-week check-in call
4. Close onboarding ticket, open ongoing support ticket
