# Crypto Wallet Setup — Ledger Nano X

Setting up a hardware wallet for long-term crypto storage. Moving assets off the exchange.

## Initial Setup

1. Downloaded Ledger Live from ledger.com (verified SHA256 checksum)
2. Connected Nano X via USB, firmware updated to 2.3.0
3. Created new wallet, generated recovery phrase
