# Obsidian Plugins Worth Trying in 2026

A curated list of plugins that have genuinely improved my workflow this year.

## 1. Dataview

Query your notes like a database. Create tables, lists, and task views from frontmatter and inline fields. Essential for anyone with more than 200 notes.

## 2. Templater

Go beyond core templates with JavaScript-powered dynamic content. Automatically insert dates, prompt for inputs, and generate boilerplate.

## 3. Kanban

Turn any note into a Kanban board. Drag-and-drop cards between columns. Syncs with regular Markdown so your data is never locked in.

## 4. Calendar

Visual calendar view linked to your daily notes. Click any date to create or open that day's note. Simple but indispensable.

## 5. Linter

Auto-format your Markdown on save. Consistent headings, spacing, and YAML frontmatter without thinking about it.
